Memory code

Este proyecto es un juego de enseñanza, pensado para aquellos usuarios que deseen aprender conceptos básico de programación de una forma entretenida, a través de este juego de memoria.

Tipo de Usuario

Va dirigido a usuarios mayores de 15 años, en particular a esos usuarios que deseen aprender los conceptos básicos de programación a través de un juego.

Qué resuelve el producto

Con esta web el usuario puede repasar o aprender-jugando los conceptos básicos de HTML,CSS y JAVASCRIPT.
